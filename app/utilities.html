Lodash

WebDriverIO doesn't bundle lodash, but you can easily add it to your project.

import _ from 'lodash'

// Example with mock response
const response = { body: [
  { id: 1, name: 'User 1' },
  { id: 2, name: 'User 2' },
  { id: 3, name: 'User 3' },
  { id: 4, name: 'User 4' }
]}

const ids = _.chain(response.body).map('id').take(3).value()

expect(ids).toEqual([1, 2, 3])

Element Selection

WebDriverIO provides powerful selectors to find elements on the page.

// Select first list item
const listItem = await $('.utility-jquery li:first-child')

// Check class and interact
await expect(listItem).not.toHaveElementClass('active')
await listItem.click()
await expect(listItem).toHaveElementClass('active')

Working with Blobs

You can use the File and Blob Web APIs directly in WebDriverIO tests.

// Example of working with blobs
const blob = new Blob(['Hello, world!'], { type: 'text/plain' })
const url = URL.createObjectURL(blob)

// Example of creating an image from a data URL
const img = await $('.utility-blob')
await browser.execute((el, imgUrl) => {
  const img = document.createElement('img')
  img.src = imgUrl
  el.appendChild(img)
}, img, '/assets/img/javascript-logo.png')

// Interact with the image
const image = await $('.utility-blob img')
await image.click()
await expect(image).toHaveAttribute('src', '/assets/img/javascript-logo.png')

Minimatch

You can use the minimatch package for glob pattern matching in your tests.

import minimatch from 'minimatch'

// Example of using minimatch
const isMatch = minimatch('/users/1/comments', '/users/*/comments', {
  matchBase: true,
})

expect(isMatch).toBe(true)

// Example with array filtering
const files = ['test.js', 'test.spec.js', 'utils.js']
const testFiles = files.filter(file => minimatch(file, '*.spec.js'))
expect(testFiles).toEqual(['test.spec.js'])

Promises

WebDriverIO uses native JavaScript Promises, making async/await syntax available.

// Using async/await with WebDriverIO
let waited = false

async function waitOneSecond() {
  return new Promise((resolve) => {
    setTimeout(() => {
      waited = true
      resolve('foo')
    }, 1000)
  })
}

describe('Promise Example', () => {
  it('should wait for promise to resolve', async () => {
    const result = await waitOneSecond()
    
    expect(result).toBe('foo')
    expect(waited).toBe(true)
    
    // Example with element interaction
    const button = await $('button')
    await button.click()
    await expect(button).toHaveText('Clicked!')
  })
})
