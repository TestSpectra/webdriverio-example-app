Misc

Examples of miscellaneous commands in WebDriverIO, for a full reference of commands, go to webdriver.io

Breaking Command Chains

In WebDriverIO, you can break a command chain by storing elements in variables and re-querying when needed.

const table = await $('.misc-table')
const cheryl = await table.$('td*=Cheryl')
await cheryl.click()

// Re-query the table to find the next element
const charles = await $('.misc-table').$('td*=Charles')
await charles.click()
Table
User: Cheryl
User: Charles
User: Darryl

exec

To execute a system command, you can use Node.js's built-in exec function.

import { exec } from 'child_process'

exec('echo Jane Lane', (error, stdout, stderr) => {
  if (error) {
    console.error(`exec error: ${error}`)
    return
  }
  expect(stdout).toContain('Jane Lane')
})

// You can use Node.js's os module to get platform information
console.log(`Platform ${process.platform} architecture ${process.arch}`)

if (process.platform === 'win32') {
  exec('type wdio.conf.js', (error, stdout, stderr) => {
    expect(stderr).toBe('')
  })
} else {
  exec('cat wdio.conf.js', (error, stdout, stderr) => {
    expect(stderr).toBe('')
  })

  exec('pwd', (error, stdout, stderr) => {
    expect(error).toBe(null)
  })
}

browser.getActiveElement()

To get the DOM element that has focus, use the browser.getActiveElement() command.

const nameInput = await $('.misc-form #name')
await nameInput.click()
let activeElement = await browser.getActiveElement()
await expect(activeElement).toHaveId('name')

const descriptionInput = await $('.misc-form #description')
await descriptionInput.click()
activeElement = await browser.getActiveElement()
await expect(activeElement).toHaveId('description')

browser.saveScreenshot()

To take a screenshot, use the browser.saveScreenshot() command.

await browser.saveScreenshot('./my-image.png')
./my-image.png

Working with objects

In WebDriverIO, you can work with objects directly without wrapping them.

const obj = { foo: 'bar' }
expect(obj).toHaveProperty('foo', 'bar')
{foo: bar}
