Viewport

Examples of changing the size of your application's viewport in WebDriverIO, for a full reference of commands, go to webdriver.io

Set Window Size

To set the viewport size and dimension, use the browser.setWindowSize() method.

// Check navbar is visible
const navbar = await $('#navbar')
await expect(navbar).toBeDisplayed()

// Set small viewport (mobile)
await browser.setWindowSize(320, 480)

// the navbar should have collapse since our screen is smaller
await expect(navbar).not.toBeDisplayed()
const toggle = await $('.navbar-toggle')
await expect(toggle).toBeDisplayed()
await toggle.click()

// Set large viewport
await browser.setWindowSize(2999, 2999)

// Common device sizes
const deviceSizes = [
  { name: 'macbook-15', width: 1440, height: 900 },
  { name: 'macbook-13', width: 1280, height: 800 },
  { name: 'ipad-2', width: 768, height: 1024 },
  { name: 'iphone-6', width: 375, height: 667 },
  { name: 'iphone-5', width: 320, height: 568 }
]

for (const device of deviceSizes) {
  await browser.setWindowSize(device.width, device.height)
  await browser.pause(200)
}

// Maximize window
await browser.maximizeWindow()
