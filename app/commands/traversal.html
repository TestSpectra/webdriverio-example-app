Traversal

Examples of traversing DOM elements in WebDriverIO, for a full reference of commands, go to webdriver.io

element.$$()

To get children of DOM elements, you can use the element.$$() command.

const breadcrumb = await $('.traversal-breadcrumb')
const activeChild = await breadcrumb.$('.active')
await expect(activeChild).toHaveTextContaining('Data')

element.$() with '..'

To get the closest ancestor DOM element, you can use element.$() with a `..` selector.

const badge = await $('.traversal-badge')
const closestUl = await badge.$('..')
await expect(closestUl).toHaveClass('list-group')

element.$$() and array index

To get a DOM element at a specific index, you can use element.$$() and access the element by its index in the returned array.

const listItems = await $$('.traversal-list>li')
await expect(listItems[1]).toHaveTextContaining('siamese')

element.$$() with filter

To get DOM elements that match a specific selector, you can use element.$$() and then filter the results.

const navItems = await $$('.traversal-nav>li')
const activeItem = navItems.find(item => item.getAttribute('class').then(c => c.includes('active')))
await expect(activeItem).toHaveTextContaining('About')

element.nextElement()

To get the next sibling DOM element, you can use the element.nextElement() command.

const apples = await $('.traversal-ul').$('li*=apples')
const oranges = await apples.nextElement()
await expect(oranges).toHaveTextContaining('oranges')

Looping through siblings

To get all of the next sibling DOM elements, you can loop through the siblings of an element.

const oranges = await $('.traversal-next-all').$('li*=oranges')
let nextSiblings = 0
let next = await oranges.nextElement()
while (next) {
  nextSiblings++
  next = await next.nextElement()
}
expect(nextSiblings).toBe(3)

Looping through siblings until a condition is met

To get all of the next sibling DOM elements until another element, you can loop through the siblings and stop when a condition is met.

const veggies = await $('#veggies')
let nextSiblings = 0
let next = await veggies.nextElement()
while (next && (await next.getAttribute('id')) !== 'nuts') {
  nextSiblings++
  next = await next.nextElement()
}
expect(nextSiblings).toBe(3)

Filtering elements

To remove DOM element(s) from a set of elements, you can filter the array of elements.

const buttons = await $$('.traversal-disabled .btn')
const enabledButton = buttons.find(btn => btn.getAttribute('disabled').then(d => !d))
await expect(enabledButton).not.toHaveTextContaining('Disabled')

Looping through parents

To get the parent DOM elements of an element, you can loop through its parents.

const cite = await $('.traversal-cite')
let parent = await cite.parentElement()
let blockquote
while (parent) {
  if ((await parent.getTagName()) === 'blockquote') {
    blockquote = parent
    break
  }
  parent = await parent.parentElement()
}
expect(blockquote).toExist()

To get the parent DOM elements of an element until another element, you can loop through its parents and stop when a condition is met.

const active = await $('.clothes-nav').$('.active')
let parents = 0

while (parent && !(await parent.getAttribute('class')).includes('clothes-nav')) {
  parents++
  parent = await parent.parentElement()
}
expect(parents).toBe(2)

Getting all siblings

To get all sibling DOM elements of an element, you can get its parent and then get all of the children of the parent, excluding the original element.

const active = await $('.traversal-pills .active')
const parent = await active.parentElement()
const siblings = await parent.$$('li')
expect(siblings.length - 1).toBe(2)
