Cookies

Examples of managing cookies in WebDriverIO, for a full reference of commands, go to webdriver.io

Get Cookie

To get a browser cookie, use the browser.getCookie() method.

// Click button to set a cookie
const btn = await $('.set-a-cookie')
await btn.click()

// Get a specific cookie
const cookie = await browser.getCookie('token')
await expect(cookie.value).toBe('123ABC')

Get All Cookies

To get all browser cookies for the current domain, use browser.getCookies().

// Get all cookies (should be empty initially)
let cookies = await browser.getCookies()
await expect(cookies).toHaveLength(0)

// Click button to set a cookie
const btn = await $('.set-a-cookie')
await btn.click()

// Get all cookies again
cookies = await browser.getCookies()
await expect(cookies).toHaveLength(1)

// Check cookie properties
const cookie = cookies[0]
await expect(cookie.name).toBe('token')
await expect(cookie.value).toBe('123ABC')
await expect(cookie.httpOnly).toBe(false)
await expect(cookie.secure).toBe(false)

Set Cookies

To set browser cookies, use browser.setCookie().

// Get all cookies (should be empty initially)
let cookies = await browser.getCookies()
await expect(cookies).toHaveLength(0)

// Set cookies
await browser.setCookie({ name: 'key', value: 'value' })
await browser.setCookie({ name: 'key2', value: 'value2' })

// Get all cookies
cookies = await browser.getCookies()
await expect(cookies).toHaveLength(2)

// Check cookie properties
await expect(cookies[0].name).toBe('key')
await expect(cookies[0].value).toBe('value')
await expect(cookies[1].name).toBe('key2')
await expect(cookies[1].value).toBe('value2')

Set Single Cookie

To set a single browser cookie, use browser.setCookie().

// Get all cookies (should be empty initially)
let cookies = await browser.getCookies()
await expect(cookies).toHaveLength(0)

// Set a cookie
await browser.setCookie({ name: 'foo', value: 'bar' })

// Get the cookie
const cookie = await browser.getCookie('foo')
await expect(cookie.value).toBe('bar')

Delete Cookie

To delete a specific browser cookie, use browser.deleteCookie().

// Cookie should not exist initially
let cookie = await browser.getCookie('token')
await expect(cookie).toBeUndefined()

// Click button to set a cookie
const btn = await $('.set-a-cookie')
await btn.click()

// Verify cookie exists
cookie = await browser.getCookie('token')
await expect(cookie.value).toBe('123ABC')

// Delete the cookie
await browser.deleteCookie('token')

// Verify cookie is deleted
cookie = await browser.getCookie('token')
await expect(cookie).toBeUndefined()

Delete All Cookies

To delete all browser cookies for the current domain, use browser.deleteAllCookies().

// Get all cookies (should be empty initially)
let cookies = await browser.getCookies()
await expect(cookies).toHaveLength(0)

// Click button to set a cookie
const btn = await $('.set-a-cookie')
await btn.click()

// Verify cookie exists
cookies = await browser.getCookies()
await expect(cookies).toHaveLength(1)

// Delete all cookies
await browser.deleteAllCookies()

// Verify all cookies are deleted
cookies = await browser.getCookies()
await expect(cookies).toHaveLength(0)

Cookie Management

Complete cookie management workflow with WebDriverIO.

// Set multiple cookies
await browser.setCookie({ name: 'key1', value: 'value1' })
await browser.setCookie({ name: 'key2', value: 'value2' })

// Get all cookies
let cookies = await browser.getCookies()
await expect(cookies).toHaveLength(2)

// Delete a specific cookie
await browser.deleteCookie('key1')

// Verify deletion
cookies = await browser.getCookies()
await expect(cookies).toHaveLength(1)
await expect(cookies[0].name).toBe('key2')

// Delete all remaining cookies
await browser.deleteAllCookies()
cookies = await browser.getCookies()
await expect(cookies).toHaveLength(0)
