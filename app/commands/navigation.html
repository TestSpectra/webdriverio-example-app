Navigation

Examples of navigating to pages within your application in WebDriverIO, for a full reference of commands, go to webdriver.io

Browser Navigation

To navigate back or forward in the browser's history, use WebDriverIO browser methods.

// Get current URL
let url = await browser.getUrl()
await expect(url).toContain('navigation')

// Navigate back
await browser.back()
url = await browser.getUrl()
await expect(url).not.toContain('navigation')

// Navigate forward
await browser.forward()
url = await browser.getUrl()
await expect(url).toContain('navigation')

// Refresh the page
await browser.refresh()

Reload Page

To reload the page, use the refresh method.

// Reload the page
await browser.refresh()

// Navigate to the same URL (hard refresh)
const url = await browser.getUrl()
await browser.url(url)

Navigate to URL

To navigate to a page, use the url method.

// Navigate to a URL
await browser.url('http://localhost:8080/commands/navigation')

// Navigate to a relative path
await browser.url('/commands/navigation')

// Wait for page to load
await browser.url('http://localhost:8080/commands/navigation')
await browser.waitUntil(
  async () => {
    const readyState = await browser.execute(() => document.readyState)
    return readyState === 'complete'
  },
  { timeout: 5000 }
)
