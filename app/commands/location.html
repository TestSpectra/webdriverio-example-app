Location

Examples of get the url within your application in WebDriverIO, for a full reference of commands, go to webdriver.io

Get URL Hash

To get the current URL hash, use the browser.getUrl() method.

const url = await browser.getUrl()
const hash = url.split('#')[1] || ''
await expect(hash).toBe('')

Get Location Object

To get the window location object, use browser.execute() to access it.

const location = await browser.execute(() => window.location)

await expect(location.href).toBe('http://localhost:8080/commands/location')
await expect(location.host).toBe('localhost:8080')
await expect(location.hostname).toBe('localhost')
await expect(location.origin).toBe('http://localhost:8080')
await expect(location.pathname).toBe('/commands/location')
await expect(location.port).toBe('8080')
await expect(location.protocol).toBe('http:')
await expect(location.search).toBe('')

Get Current URL

To get the current URL, use the browser.getUrl() method.

const url = await browser.getUrl()
await expect(url).toBe('http://localhost:8080/commands/location')
